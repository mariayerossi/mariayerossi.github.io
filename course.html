Simple Online Course Mobile Application

This is a mobile computing practicum project that I worked on in the 5th semester. In this application, users can register and login as a teacher or a student. Teachers can add classes by filling in the details of the class and students can join the class by pressing the class button. When entering this application, at the bottom there will be a bottom navigation that, when pressed, will switch between one fragment and another. In addition, user and class data in this application are stored in local storage.

Class : Mobile Computing (Practicum)
