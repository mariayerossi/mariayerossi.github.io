Simple Point of Sale Application

A POS application for bread purchases. This application will display a login page, there are 2 roles in this application, namely employees and chefs. When a user logs in as an employee, they will be directed to the bread purchase transaction page (first screenshot image), on the transaction page, employees can add items to the cart and can clear the cart. However, if a user logs in as a chef, they will be directed to the second page (second screenshot image), which is the recipe creation page. Chefs can add, update, and delete recipes.
